# Pet Shop Frontend

## Introduction
This is the frontend application for the Pet Shop built using Vue.js.

## Prerequisites
- Node.js
- npm

## Installation

### Step 1: Clone the repository
```sh
- ssh git clone https://github.com/Dante983/pet-shop-ui.git
- cd pet-shop-frontend


- npm install
- cp .env.example .env
- npm run dev
```
